# OSTROV Website

Это лендинг-визитка для компании OSTROV (VOSTROVA LLC) с поддержкой трёх языков, эффектом Glassmorphism и контактной формой с отправкой заявок прямо в Telegram.

## Подключение Telegram-бота к форме (Server-Side)

Чтобы контактная форма безопасно и корректно отправляла заявки в вашу личную «личку» или группу Telegram, следуйте этой короткой инструкции:

### 1. Как получить Bot Token
1. Откройте Telegram и найдите официального бота **@BotFather** (с синей галочкой).
2. Напишите ему команду `/newbot`.
3. Придумайте название и username для вашего бота (например, `OstrovFeedbackBot`).
4. В ответ вы получите длинный токен (вида `1234567890:AAH_XXXXXXXXXXXXX_XXXXXXX`). Обязательно сохраните его.

### 2. Как получить Chat ID (куда будут приходить заявки)
**Если вы хотите получать заявки в личные сообщения:**
1. Найдите бота **@userinfobot** и напишите ему `/start`.
2. Он пришлет ваш личный числовой `Id` (например, `123456789`). Это и есть ваш Chat ID.
3. **ВАЖНО:** Найдите в поиске вашего только что созданного бота и **нажмите кнопку "Запустить"** (Start). Если этого не сделать, бот не сможет вам писать!

**Если вы хотите получать заявки в группу (много администраторов):**
1. Создайте группу в Telegram и добавьте туда вашего нового бота.
2. Временно добавьте в эту же группу бота **@RawDataBot**. Он в ту же секунду пришлет системную информацию в чат, в строке `"chat": {"id": -100XXXXXXXXXX` будет ID вашей группы.
3. Скопируйте этот ID вместе с минусом (это ваш Chat ID). Бота `@RawDataBot` после этого из группы можно удалить.

### 3. Настройка на сервере (Хостинг)
В корне вашего сайта есть файл `prod.send.php`. 
Это защищенная от утечек «заготовка» скрипта отправки.\n
1. Загрузите файлы сайта на ваш хостинг.\n
2. Переименуйте `prod.send.php` на сервере в **`send.php`**.\n
3. Откройте `send.php` (для редактирования прямо на сервере) и замените две переменные:

```php
// Вставьте ваш реальный Bot Token, полученный от @BotFather
$TELEGRAM_BOT_TOKEN = '1234567890:AAH_XXXXXXXXX';

// Вставьте ваш Chat ID
$TELEGRAM_CHAT_ID = '-1001234567890';
```
4. Сохраните изменения. 

Все готово! Форма заявок теперь полностью функциональна и 100% безопасна (токены скрыты от чужих глаз на бэкенде).

## Деплой
Обратите внимание, что файл `send.php` с вашими секретными ключами **уже добавлен в исключения (.gitignore)**. Это значит, что ваши ключи ни при каких условиях случайно не попадут в открытый доступ на GitHub. Для загрузки кода в интернет всегда используйте файл болванку `prod.send.php`.
